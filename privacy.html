Kebijakan Privasi

Rabbit Hole Mindmap Learner

Terakhir diperbarui: 29 Mei 2026

1. Pengantar

Selamat datang di Rabbit Hole Mindmap Learner (lubangkelinci.my.id). Privasi Anda adalah prioritas kami. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data Anda saat menggunakan layanan kami.

Dengan menggunakan aplikasi ini, Anda menyetujui praktik yang dijelaskan dalam kebijakan ini. Jika Anda tidak setuju, Anda dapat menolak analytics melalui banner cookie atau pengaturan privasi.

2. Data yang Kami Kumpulkan

2.1 Data Esensial (wajib, tanpa consent)

  • Session cookie — disimpan jika Anda login via Google OAuth, untuk menjaga sesi login tetap aktif. Bersifat essential dan tidak memerlukan consent.
  • Data mindmap — topik, node, dan progress belajar Anda. Disimpan agar sesi belajar Anda bisa dilanjutkan.

2.2 Data Analytics (opsional,但uh consent)

Data berikut hanya dikumpulkan jika Anda menyetujui cookie analytics:

  • Node events — nama node yang Anda buka, durasi baca, jumlah token AI yang digunakan, dan model AI yang dipakai.
  • Session events — durasi sesi online (berapa lama Anda membuka aplikasi), pembuatan mindmap, dan hasil quiz.
  • Preferensi aplikasi — tema (terang/gelap), bahasa, dan gaya penulisan yang dipilih.
🛡️ Kami TIDAK mengumpulkan:
Alamat IP, lokasi geografis, data perangkat, riwayat browser di luar aplikasi, atau informasi identitas pribadi lainnya (selain email Google jika Anda login).

3. Bagaimana Data Digunakan

Data yang kami kumpulkan digunakan untuk:

  • Menyediakan dan meningkatkan pengalaman belajar mindmap interaktif
  • Menganalisis fitur mana yang paling bermanfaat bagi pengguna
  • Memantau biaya token AI (untuk pengembangan internal)
  • Menampilkan statistik belajar pribadi Anda di dashboard

4. Penyimpanan & Keamanan

  • Data disimpan di PostgreSQL database melalui layanan hosting yang aman.
  • Kami tidak membagikan data Anda kepada pihak ketiga untuk tujuan komersial atau periklanan.
  • Koneksi ke server menggunakan HTTPS untuk mengenkripsi data selama transmisi.
  • Password database dan API key disimpan di environment variables, tidak pernah terekspos ke klien.

5. Cookie

Cookie yang kami gunakan:

Jenis Cookie Tujuan Kategori
session_id Menyimpan sesi login (Google OAuth) Essential
rabbit_hole_consent Preferensi consent analytics Anda Essential
app_theme, ai_language, dll. Preferensi pengaturan aplikasi (localStorage) Essential

6. Hak Anda

Berdasarkan Undang-Undang Perlindungan Data Pribadi (UU PDP) Indonesia, Anda memiliki hak untuk:

  1. Menolak pengumpulan data analytics kapan saja (via banner cookie atau Settings → Privasi)
  2. Menghapus sesi dan data belajar Anda dengan logout atau menghapus mindmap
  3. Mengakses — lihat data yang kami kumpulkan melalui Dashboard → Stats
  4. Menarik consent — matikan toggle analytics di Settings, dan kami akan berhenti melacak

7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan signifikan akan diumumkan melalui aplikasi. Tanggal "Terakhir diperbarui" di bagian atas menunjukkan kapan kebijakan ini terakhir direvisi.
